# XM-Take-Home-Task

A photo gallery SPA with an infinite scrolling photostream and a favorites system. Built as a take-home task for a Senior Angular Developer role at XM. Uses Angular 22, Angular Material, and the picsum.photos API.

## Getting Started

```bash
  git clone https://github.com/SieczkoIT/XM-Take-Home-Task.git
  cd XM-Take-Home-Task
  npm install
  ng serve
```

Open http://localhost:4200.

## Running Tests

```bash
  ng test
```

Uses Vitest as the test runner. Every component, service, and directive has unit tests.
